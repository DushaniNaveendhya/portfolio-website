# Dushani Naveendhya - Portfolio

Welcome to my personal portfolio website! This site showcases my skills, projects, and contact information in a modern, responsive design.

## Features

- Responsive design that works on all devices
- Modern UI with smooth animations
- Project showcase section
- Skills highlighting
- Contact information
- Smooth scrolling navigation

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Grid Layout

## Sections

- **Header**: Professional header with profile placeholder and social links
- **About**: Personal introduction and skills showcase
- **Projects**: Highlighted projects with descriptions
- **Contact**: Contact information and call to action

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content with your own information
4. Add your own projects and images

## Customization

To customize this portfolio for your own use:

1. Update the name and title in `index.html`
2. Replace the placeholder profile with your own image
3. Update the "About Me" section with your own bio
4. Modify the skills list to match your expertise
5. Add your own projects to the projects section
6. Update contact information
7. Add links to your social profiles

## Files

- `index.html`: Main HTML structure
- `styles.css`: All styling
- `script.js`: Interactive functionality
